# Template

This project is a **Next.js (App Router)** starter template for building a multilingual web app with **TypeScript** and **Tailwind CSS**.

It includes a clean base structure with reusable UI components, language switching via context, and i18n dictionaries for English and Vietnamese.

## Summary

- Built with Next.js, React, TypeScript, and Tailwind CSS
- Uses App Router structure under `src/app`
- Includes shared components such as header and action buttons
- Provides language state management through `LanguageContext`
- Ships with i18n resources in `src/i18n` (`en`, `vi`)
- Contains starter pages including home and help routes
